# Règles de base Gaslands

Gaslands est un jeu de figurines post-apocalyptique de **course automobile armée**, publié par Osprey Games. Les joueurs s'affrontent dans des arènes déchirées à bord de véhicules modifiés jusqu'aux dents.

## Le principe du jeu

Chaque joueur contrôle une **équipe** composée de plusieurs véhicules. L'objectif dépend du scénario joué (course, massacre, livraison...).

## Le tour de jeu

Le jeu se déroule en **activations alternées**. À chaque activation, un véhicule peut :

1. **Se déplacer** en utilisant des gabarits de mouvement
2. **Attaquer** les véhicules adverses à portée
3. **Utiliser des capacités spéciales** (nitro, mines, grappin...)

## Les dés d'attaque

Gaslands utilise des **dés spéciaux** (ou des d6 standard avec une table de correspondance) :

- 🎯 **Hit** — Touche l'adversaire
- 💥 **Smash** — Touche + dommages supplémentaires
- 💀 **Skull** — Touche critique
- ⚙️ **Gear** — Compte dans certaines capacités spéciales
- ✨ **Ammo** — Résultat spécial selon l'arme

## Les sponsors

Chaque équipe choisit un **sponsor** qui définit les véhicules et armes disponibles :

| Sponsor | Spécialité |
|---------|-----------|
| Rutherford | Équipes polyvalentes |
| Miyazaki | Vitesse et agilité |
| Mishkin | Technologies étranges |
| Verney | Puissance de feu |
| Idris | Survie et robustesse |
| Warden | Compétitions officielles |

## Le budget

Les équipes sont construites avec un budget en **jerricans** (🥫). Le budget standard est de **50 jerricans** pour une partie d'initiation.
